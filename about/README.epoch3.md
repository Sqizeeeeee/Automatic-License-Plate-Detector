

# Добавление нового функционала

## Разделение на блоки

Добалено разделение на модули: detector, ocr_engine, image_processor, video_processor. video_processor использует ocr_engine и detector также как и imgae_processor.


## Новый блок: Video Processor

- Добавлен модуль video_processor для обработки видеофайлов.
- VideoProcessor извлекает кадры из видео, применяет детектор и OCR к каждому N-му кадру
- Собираются все уникальные номера


## Прочие улучшения

- Повышена стабильность: бот корректно обрабатывает ошибки и очищает временные файлы, добавлено логирование.


## Основные проблемы системы на данный момент

- Слабая работа модели по определению области номера. Под резким углом, ночью или в других не обычных условия модель не находит область с номером. 
- OCR работает хорошо, ошибки встречаются редко
